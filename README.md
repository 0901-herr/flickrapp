
# FlickrApp

## Description
Welcome to the FlickrApp website! This is a web application that could be used to search for matching images from Flickr. This project is built using MongoDB, Express.js, Node.js, and React.js. 

## Local Installation: How to run locally on your computer
To run the web application on your computer, please follow these steps:
1. Log in to MongoDB with the provided credentials (https://cloud.mongodb.com/v2/64a24d0efcdeb74d4f812e3e#/security/network/accessList)
  
2. Add your IP address at the 'Network Access' tab

3. Ensure that mongodb is installed on your machine, if not run
```
npm install mongodb
```  
 
4. Once MongoDB is setup, In the backend folder, run
```
nodemon server
```

5. In the frontend folder, run
```
npm start
```

6. The application should be running on your localhost 3000
